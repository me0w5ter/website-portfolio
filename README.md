# Harsh Dogra | Software Engineering Portfolio

A modern, responsive personal portfolio website showcasing my journey as a Software Engineering student at Curtin University. This project features a custom-built AI Intent Engine, a "Glassmorphism" UI, and dynamic background animations.


## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3 (Custom Glassmorphism), JavaScript (ES6+)
- **Framework:** Bootstrap 5.2 (Customised)
- **Icons:** Bootstrap Icons
- **Deployment:** GitHub Pages

## ✨ Key Features
- **AI Assistant:** A custom-built JavaScript Intent Engine that answers questions about my skills, projects, and education in real-time.
- **Dynamic UI:** Features a "floating code" background that randomly shuffles technical snippets (Java, Python, Docker, etc.) for a developer-centric aesthetic.
- **Modern Navigation:** Sticky "Glassmorphism" navbar with backdrop-filter effects.
- **Project Showcase:** Interactive cards for my **Anti-Virus Scanner** (Python/PyQt5) and **AI Resume Platform** (React/Spring Boot).
- **Fully Responsive:** Optimised for desktop, tablet, and mobile viewing.
